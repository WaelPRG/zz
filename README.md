# Roblox Simulator Core - Technical Portfolio

A modular technical foundation for a Roblox simulator game. This project serves as a showcase of advanced Roblox development workflows and clean architecture, rather than a finished product.

## 🛠 Tech Stack
- **Framework:** [Knit](https://sleitnick.github.io/Knit/) (Service-Controller architecture).
- **Tooling:** [Rojo](https://rojo.space/) (External coding in VS Code & Git version control).
- **Data Persistence:** [ProfileStore](https://laminastudios.github.io/ProfileStore/) (Secure data handling).
- **Language:** Luau (Type-checked Lua).

## 🚀 Key Features
- **Material Spawner:** Automated system for procedural resource generation.
- **Data Management:** Robust player profile system with auto-saving and session locking.
- **Modular Design:** Highly organized codebase split into Services, Controllers, and Shared modules.

## 📌 Note
This repository is a **technical showcase** of my ability to implement scalable systems in Roblox using industry-standard tools.
